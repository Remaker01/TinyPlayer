### 简易播放器TinyPlayer
采用QT构建的音频播放器

快捷键
+ Ctrl+O：打开音乐文件
+ Ctrl+Shift+O：打开目录，自动添加并扫描目录下的合法文件
+ Alt+O：打开背景图片文件
<hr>

1. 支持格式：采用的播放器引擎是QMediaPlayer,支持的格式主要与平台有关，Windows下默认支持mp3,wav,wma,aiff格式。
2. 构建平台：
   1. 硬件与系统环境：Intel 第十代酷睿CPU+Windows 10 x64
   2. 软件平台：QT5.12+QT Creator 5+CMake 3.21+MinGW 8.1
